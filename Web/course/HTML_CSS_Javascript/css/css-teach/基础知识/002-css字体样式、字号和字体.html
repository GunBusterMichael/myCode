企业简介

公司由来

华为技术em有限公司，成立于1987年，总部位于em广东省深圳市龙岗区。 华为是全球领先的信息与通信技术（ICT）解决方案供应商。

公司发展

专注于ICT领域，坚持稳健经营、持续创新、开放合作，在电信em运营商、企业、终端和云计算等领域构筑了端到端的解决方案优势，为运营商客户、企业客户和消费者提供有竞争力的ICT解决方案、产品和服务，并致力于实现未来信息社会、构建更美好的全联接世界。

公司地位

2013年，华为首超全球第一大电信设备商爱em立信，排名《财富》世界500强第315位。华为的产品和解决方案已经应用于全球170多个国家，服务全球运营商50强中的45家及全球1/3的人口。

2020年8月10日，《财富》公布世界em500强榜（企业名单），华为排在第49位。2020中国民营企业500强第一名。
