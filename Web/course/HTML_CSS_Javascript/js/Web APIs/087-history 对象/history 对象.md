# history 对象

## 1. 定义

- history 对象由 window 对象提供；
- history 对象可以与浏览器浏览历史记录进行交互；
- history 对象包含用户在浏览器窗口中访问过的 URL。

## 2. 方法

| history 对象方法 | 功能                                    |
| ---------------- | --------------------------------------- |
| back()           | 后退                                    |
| forward()        | 前进                                    |
| go(参数)         | 前进或后退。参数等于2：前进两页；参数等于-2：后退两页 |

## 3. 使用场景

一般开发中少见，==OA 系统==中常见。
