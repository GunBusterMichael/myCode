美国75周印钞31万亿,伊朗正式用人民币替代美元后,意料外的事发生

发布时间: 11-12 07:10 BMC中文网 评论中大奖（11人参与） | 收藏本文

据伊朗海关总署 (IRICA) 【百度百科】在11月8日公布的数据显示，截止10月22日的一个月内，伊朗非原油出口创下历史新高，达到52.68亿美元，海外出货量比9月增长了27%以上，比前五个月的平均水平增长了 60%，因为该国不断鼓励非原油出口活动的增加，以抵消因美国经济制裁造成的石油出口损失。

另据伊朗金融论坛报在10月30日援引伊朗海关总署（IRICA）最新公布的数据显示，伊朗运往上海合作组织国家的货物价值超过95.58亿美元，在截止9月底的六个月中，伊朗对该组织成员国的出口额增长了61%，总计超过2280万吨，伊朗预计其与上合组织的贸易将显着增加。

报告数据显示，从伊朗【百度百科】出口的主要商品和产品包括液化天然气、甲醇、聚乙烯、钢铁产品、石化产品和汽油，中国是伊朗商品和产品的最大客户，采购额为65亿美元，其次是伊拉克和土耳其，另外，运往阿联酋和阿富汗的货物也分别达到22亿和10亿美元。2021年第二季度，伊朗对东南亚国家联盟的出口增长了48%至11亿美元。

伊朗多年来一直是该组织的观察成员国，旨在扩大与中国等主要成员的贸易关系，同时寻求加强与俄罗斯和中亚地区内陆国家的关系，同时，伊朗可以通过其庞大的交通网络为上合组织成员与区域外经济体之间的贸易合作提供便利的独特机会，对中国的出口占伊朗对上合组织出口总额的三分之二以上。

据伊朗金融论坛报在11月8日援引伊朗贸易促进组织 (TPO)【百度百科】相关负责人的最新讲话称，伊朗和巴基斯坦双方同意在未来三个月内消除两国之间自由贸易（协议）面临的障碍和问题，并已经同意在未来两年内将年度贸易额增加五倍，以达到50亿美元的目标，紧接着，伊朗和巴基斯坦签署了他们的第一个以货易货的贸易协议，用伊朗液化石油气交换巴基斯坦的大米。

这标志着两个相邻大国之间的贸易实行绕开美元而进行的一次成功举措，在贸易和投资方面，伊朗受到美国最严厉经济和金融制裁的影响，这迫使伊朗及其贸易伙伴在缺乏国际银行渠道的情况下寻找替代创新方法。

而就在一周前（10月26日），伊朗和叙利亚也将于11月底签署自由贸易协定，同时伊朗还将接下去与巴基斯坦、印尼和委内瑞拉加强贸易合作和相关优惠协定的落实。
