暖消息！这些地方气温将回升

中国青年网 发布时间: 11-11 12:37中国青年网官方帐号

随着寒潮影响结束，今起三天（11月11日至13日）全国大部地区降水稀少，只有黑龙江东北部局地今天有中到大雪，明天东北地区的降雪将彻底结束。气温方面，全国大部地区气温将逐步缓慢回升。

11月4日至9日，今年下半年以来的最强寒潮自西北向东南影响我国大部，此次寒潮降温幅度大、雨雪范围广、影响大。有166个国家级气象站日最低气温创11月上旬历史同期最低，辽宁、吉林、黑龙江、内蒙古、河北、河南、山东、天津等地有151个国家气象站日降水量突破11月历史极值

昨天，这股寒潮的影响趋于结束，大部地区气温逐渐回升，降水较少。未来几天，我国大部地区被干燥的偏北气流控制，晴朗少云的天气成为主流。为数不多的降水分布在西南地区和东北地区，西南地区多是小雨或阵雨天气，黑龙江东部、吉林东部、辽宁北部等地今天有小雪，黑龙江东北部局地有中到大雪，明天东北地区的降雪将彻底结束。

气温方面，未来一周，全国大部地区气温将逐步回升，但由于受到冷高压控制，还会不断有弱冷空气影响，因此气温回升较为缓慢。东北地区最高气温在冰点徘徊，华北等地最高气温在10℃左右，像北京、天津，未来一周最高气温在10℃上下，最低气温0℃左右，在常年正常水平附近波动。江南气温也有明显回升，最高气温将接近或者达到20℃。
