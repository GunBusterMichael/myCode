1986年，刘志远以吉他手和键盘手的身份加入Beyond；之后，他们和小岛乐队、达明一派合录了一盘盒带《劲歌金曲》，并收录了他们创作的4首歌曲。4月，Beyond自己出资举办香港中环演艺会。同年，叶世荣和黄贯中、邓炜谦、马永基组成一支名为“高速啤机”的重金属乐队；该乐队是Beyond的分支乐队，并以玩票性质参加了一些地下音乐会的演出。年底，正式签约Kinn's Music Ltd 。 1987年，发行EP《永远等待》；其中同名歌曲《永远等待》是Beyond尝试用粤语创作摇滚作品的开端，而EP中的另外一首歌曲《昔日舞曲》则登上了香港电台流行榜，并且成为Beyond第一首被电视台拍成MTV的歌曲。同年，发行带有中东风情的粤语专辑《亚拉伯跳舞女郎》，并尝试寻找介于商业和摇滚间的平衡点，专辑中的同名歌曲获得香港电台龙虎榜周冠军，歌曲《昔日舞曲》再次被收录在该专辑中，并在香港电台龙虎榜上排名第8位。10月4日，在高山剧场举办“Beyond超越亚拉伯”演唱会。 1988年，签约新艺宝唱片公司。3月，发行粤语专辑《现代舞台》，该专辑不仅是Beyond批判社会现象的开端，而且音乐走向相比Beyond以往专辑更显柔和，黄贯中和黄家强首次拥有自己的配唱歌曲，开始兼任乐队的主唱；成员们也把头发剪短，一洗反叛青年的形象。4月，Beyond在大专会堂举办了一场音乐会，在音乐会上，刘志远宣布退出Beyond，乐队成员再次恢复到四人。
