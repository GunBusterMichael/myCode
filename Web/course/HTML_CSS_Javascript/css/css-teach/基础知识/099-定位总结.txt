定位模式            是否脱标占有位置        是否可以使用边偏移      移动位置基准
静态static          不脱标，正常模式             不可以             正常模式
相对定位relativ     不脱标，占有位置              可以              相对自身位置移动（自恋型）
绝对定位absol       完全脱标，不占有位置           可以             相对于已定位父级位置移动（拼爹型）（无父级或父级无定位，则按照浏览器对齐）
固定定位fixed       完全脱标，不占有位置           可以             相对于浏览器位置移动
