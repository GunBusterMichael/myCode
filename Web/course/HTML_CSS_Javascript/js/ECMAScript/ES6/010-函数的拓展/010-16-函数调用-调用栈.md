# 调用栈

## 1. 调用帧

### 1.1 定义

> ==函数被调用==时，会在==内存中==形成一个“调用记录”，也被称为==“调用帧”==。
>
> “调用帧”==保存==着该函数的==调用位置==和==内部变量==。

## 2. 调用栈

### 1.2 定义

> 所有的“调用帧，组成一个“调用栈”

### 1.3 内嵌函数的调用栈

函数 A 的内部调用函数 B，那么在 A 的调用帧上方，还会形成一个 B 的调用帧。等到 B 运行结束，将结果返回到 A，B 的调用帧才会消失。如果函数B内部还调用函数 C，那就还有一个 C 的调用帧，以此类推。所有的调用帧，就形成一个“调用栈”（call stack）。
